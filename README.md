## color O' clock
A #hex color clock.

### Installation 
1. install meteor: curl https://install.meteor.com/ | sh
2. cd /clock: meteor run
3. point browser to: http://localhost:3000/

### Deployment
meteor deploy {appname}.meteor.com

### Sample
http://colorclock.meteor.com/
